# 记事本计算器

随手记录，即时计算。在文本中混合书写和数学表达式，右侧实时显示计算结果。

## 功能

- **自由文本 + 计算混排** — 像记事本一样书写，包含数学表达式的行自动计算
- **变量赋值** — `单价 = 15`，后续行可引用
- **链式运算** — 以运算符开头（如 `* 5`）自动接上上一个结果
- **`ans` 引用** — 引用上一行的计算结果
- **`#` 注释** — 以 `#` 开头的行不参与计算
- **百分比** — `20%` 自动转换为 `0.2`
- **数学函数** — `sqrt`, `pow`, `sin`, `cos`, `abs`, `ceil`, `floor`, `round`, `min`, `max`, `log`, `PI`, `E`
- **千分位格式化** — 大数字自动添加逗号分隔
- **深色/浅色主题** — 跟随系统或手动切换
- **导出** — 将计算记录导出为 `.txt` 文件
- **localStorage 持久化** — 刷新页面不丢失数据
- **响应式设计** — 支持移动端

## 技术栈

- TypeScript + Vite + Bun
- 纯原生 DOM + [mathjs](https://mathjs.org/) 按需导入计算引擎
- CSS 变量主题系统

## 开发

```bash
bun install
bun run dev
```

## 构建

```bash
bun run build
```

产物输出到 `dist/` 目录。
